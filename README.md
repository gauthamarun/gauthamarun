## Featured Projects

### 🚕 [NYC TLC Batch Data Pipeline](https://github.com/gauthamarun/nyc-trips-pipeline)
End-to-end batch pipeline over **3M+ rows** of NYC Yellow Taxi data.
- Designed a **star schema** (fact_trips, dim_location, dim_vendor) in DuckDB with Kimball modeling
- Authored SQL transformations using window functions: RANK, rolling averages, percentiles
- Orchestrated daily runs via **Apache Airflow** with retry logic and idempotent task design
- Added a **pytest data quality gate** to catch schema violations before transforms run
- `Python · DuckDB · Apache Airflow · Parquet · Star Schema · pytest`

---

### 🚗 [Driver Distraction Detection](https://github.com/gauthamarun/Driver-Distraction-Model)
CNN + GAN-based driver monitoring system — **published at IEEE Global AI Summit 2024**.
- Applied Neural Style Transfer augmentation on the 3MDAD dataset
- Used meta-learning to improve model generalization across distraction categories
- `Python · PyTorch · CNN · GAN · Neural Style Transfer · Meta-Learning`
- 📄 [Read the paper](https://doi.org/10.1109/GlobalAISummit62156.2024.10947896)
